# Semantic Car Description

We are building an e-commerce platform that sells cars.
Given a user’s description, we perform a **semantic search** to find the most similar car.

## Task

Based on the provided car features, generate an Arabic **natural and human-like description** of the car that can be used for **sentence similarity** tasks.
Make sure the description sounds realistic and engaging — for example, mention whether the car is suitable for families, sports enthusiasts, or daily commuting...
